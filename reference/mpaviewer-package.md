# mpaviewer: MPA Viewer

An interactive overview of selected biodiversity assets in Marine
Protected Areas of Western Australia.

`mpaviewer` is an interactive dashboard for Western Australian marine
monitoring data, hosted at <https://mpaviewer.dbca.wa.gov.au/>.
